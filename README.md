# Mutual Fund Analytics

## Project Overview

This project is part of the Bluestock Fintech Internship.

The objective is to perform ETL (Extract, Transform, Load), Data Cleaning, SQL Database Design, Exploratory Data Analysis (EDA), and Dashboard Development using Mutual Fund datasets.

## Tech Stack

- Python
- Pandas
- NumPy
- SQL
- Matplotlib
- Seaborn
- Plotly
- SQLAlchemy
- Jupyter Notebook
